## ðŸ“± About the App

**Clean Skate** is an unofficial utility designed for figure skaters, coaches, and fans. It allows users to:
*   Calculate **Technical Element Scores (TES)** based on ISU Scales of Values.
*   Estimate **Program Component Scores (PCS)**.
*   Plan competitive routines and analyze potential scoring outcomes.

*Disclaimer: This application is an independent project and is not affiliated with the International Skating Union (ISU) or any national federation.*
